What DevLift Offers

From aspiring developers and visual graphic designers to professionals, we have the perfect training to develop your skills.

Creative Web Design

This course will handle the designing and maintaining of the Front-End and UI (User Interface) of a website. At the end of the course, students are expected to learn the fundamentals of web designing and familiarization/expertise in the use of CSS Frameworks like Materialize and BootStrap - which are commonly used in web designing and building portfolios.

Web Development

This course refers to the back-end process of the website. This course aims to provide modules and lessons on the functionalities of a website. At the end of this course, students will have the knowledge and familiarization of the fundamentals, functions, object-oriented programming and the scripts for making a connection and transferring data between databases

Mobile Responsiveness

A module included in Creative Web Design. This course will teach students to build a mobile-friendly web application. After this course, students will be able to learn how to create responsive websites efficiently using CSS Frameworks.

Database Management

Databases are where most information found in websites are stored. This is one of the major components in a back-end process. This course aims to educate the trainees / learners on how to use databases such as MySQLi's cpanel and MongoDB's collections. At the end of this course, students will be able to create a connection to their own database, create,delete and update information stored in the database.
